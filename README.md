## IE - Biblioteca de validação de inscrições estaduais
Escrito por @renatoargh

### Utilização:

    validar(uf, inscricaoEstadual);
    
Onde, `uf` deve estar entre um dos valores 
    
    var estados =
      ["ac", "al", "ap", "am", 
      "ba", "ce", "df", "es",
      "go", "ma", "mt", "ms",
      "mg", "pa", "pb", "pr",
      "pe", "pi", "rj", "rn",
      "rs", "ro", "rr", "sc",
      "sp", "se", "to"];
